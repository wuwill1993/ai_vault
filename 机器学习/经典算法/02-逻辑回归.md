# 逻辑回归 (Logistic Regression)

> 名字叫"回归"，实际是分类算法

---

## 一、核心思想

逻辑回归本质是**线性模型 + Sigmoid函数**，将线性输出映射到[0,1]区间，表示属于正类的概率。

```
z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
p = σ(z) = 1/(1 + e^(-z))
```

- 如果 $p \geq 0.5$，预测为正类(1)
- 如果 $p < 0.5$，预测为负类(0)

---

## 二、Sigmoid函数

### 2.1 定义

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

### 2.2 重要性质

1. **值域**：$(0, 1)$，可解释为概率
2. **单调递增**：保持线性模型的有序性
3. **导数优美**：$\sigma'(z) = \sigma(z)(1 - \sigma(z))$
4. **中心对称**：$\sigma(0) = 0.5$

### 2.3 为什么用Sigmoid？

| 函数 | 优点 | 缺点 |
|------|------|------|
| Sigmoid | 输出[0,1]可解释概率 | 两端梯度趋近0 |
| Tanh | 零中心化 | 输出[-1,1] |
| ReLU | 无梯度消失 | 输出无上限 |

**核心原因**：输出可直接解释为概率！

---

## 三、损失函数

### 3.1 交叉熵损失

$$L = -\frac{1}{m}\sum_{i=1}^{m}[y^{(i)}\log(\hat{y}^{(i)}) + (1-y^{(i)})\log(1-\hat{y}^{(i)})]$$

**直观理解**：
- 当 $y=1$ 时，预测越接近1，损失越小
- 当 $y=0$ 时，预测越接近0，损失越小

### 3.2 为什么不用MSE？

1. **非凸函数**：存在多个局部最优解
2. **收敛慢**：预测错误时梯度接近零
3. **概率解释**：交叉熵衡量概率分布距离

### 3.3 与MLE的关系

交叉熵可从**最大似然估计**推导：

$$\log L(\theta) = \sum_{i=1}^{m}[y^{(i)}\log(\hat{y}^{(i)}) + (1-y^{(i)})\log(1-\hat{y}^{(i)})]$$

最大化似然 = 最小化交叉熵

---

## 四、梯度推导

**前向传播**：
```
z = w^T x + b
a = σ(z)
L = -[y·log(a) + (1-y)·log(1-a)]
```

**反向传播**：
$$\frac{\partial L}{\partial w} = (a - y) \cdot x$$

**最终梯度公式**：
$$\frac{\partial L}{\partial w_j} = \frac{1}{m}\sum_{i=1}^{m}(a^{(i)} - y^{(i)})x_j^{(i)}$$

结果优美：**误差 × 特征值**

---

## 五、多分类扩展

### 5.1 Softmax回归

$$p(y=k|x) = \frac{e^{z_k}}{\sum_{j=1}^{K}e^{z_j}}$$

### 5.2 One-vs-Rest (OvR)

每个类别训练一个二分类器，预测时选概率最高的。

### 5.3 One-vs-One (OvO)

每两个类别训练一个分类器，投票决定。需 $\frac{K(K-1)}{2}$ 个分类器。

## 六、决策边界

### 6.1 线性决策边界

$$w^T x + b = 0$$

### 6.2 非线性决策边界

通过多项式特征实现：

```python
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import Pipeline

model = Pipeline([
    ('poly', PolynomialFeatures(degree=2)),

```

## 七、优缺点

### 优点
- 简单高效，训练速度快
- 输出有概率解释
- 可解释性强
- 适合高维稀疏数据

### 缺点
- 只能处理线性可分问题
- 对特征工程依赖大
- 对多重共线性敏感
- 容易欠拟合

## 八、代码实现

### Python手写实现

```python
import numpy as np

class LogisticRegression:
    def __init__(self, learning_rate=0.01, n_iterations=1000):
        self.lr = learning_rate
        self.n_iterations = n_iterations

    def sigmoid(self, z):
        return 1 / (1 + np.exp(-np.clip(z, -500, 500)))

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0

        for _ in range(self.n_iterations):
            z = np.dot(X, self.weights) + self.bias
            a = self.sigmoid(z)

            dw = (1 / n_samples) * np.dot(X.T, (a - y))
            db = (1 / n_samples) * np.sum(a - y)

            self.weights -= self.lr * dw
            self.bias -= self.lr * db

    def predict_proba(self, X):
        return self.sigmoid(np.dot(X, self.weights) + self.bias)

    def predict(self, X, threshold=0.5):
        return (self.predict_proba(X) >= threshold).astype(int)
```

### Sklearn实现

```python
from sklearn.linear_model import LogisticRegression

model = LogisticRegression(penalty='l2', C=1.0)
model.fit(X_train, y_train)

y_pred = model.predict(X_test)
y_proba = model.predict_proba(X_test)
```

## 九、面试高频问题

### Q1: 逻辑回归是线性模型吗？

**是的**。决策边界 $w^Tx + b = 0$ 是线性的。

### Q2: 如何处理不平衡数据？

1. `class_weight='balanced'`
2. 调整阈值
3. 过采样/欠采样

### Q3: 逻辑回归 vs SVM？

| 对比 | 逻辑回归 | SVM |
|------|----------|-----|
| 输出 | 概率 | 类别 |
| 损失函数 | 交叉熵 | Hinge Loss |
| 核函数 | 无 | 有 |

### Q4: 如何解决欠拟合？

1. 增加特征（多项式、交互特征）
2. 减小正则化强度
3. 使用更复杂的模型
