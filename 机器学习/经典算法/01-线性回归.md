# 线性回归 (Linear Regression)

> 最基础的回归算法，理解机器学习的起点

---

## 一、核心思想

线性回归假设**目标变量与特征之间存在线性关系**，通过拟合一条直线（或超平面）来预测连续值。

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

其中：
- `wᵢ` 是权重（斜率）
- `b` 是偏置（截距）
- 目标是找到最优的 `w` 和 `b`

---

## 二、数学推导

### 2.1 目标函数

使用**均方误差(MSE)**作为损失函数：

$$J(w, b) = \frac{1}{m}\sum_{i=1}^{m}(y^{(i)} - \hat{y}^{(i)})^2$$

其中 $\hat{y}^{(i)} = w^Tx^{(i)} + b$ 是预测值。

**为什么用MSE？**
1. 凸函数，有全局最优解
2. 对异常值敏感（可能优点也可能缺点）
3. 数学上易于求导

### 2.2 求解方法

#### 方法一：最小二乘法（闭式解）

对 $J(w)$ 求导并令其为零：

$$\frac{\partial J}{\partial w} = 0$$

得到闭式解（假设数据矩阵为 $X$）：

$$w = (X^TX)^{-1}X^Ty$$

**优点**：一步得到最优解
**缺点**：
- 矩阵求逆计算量大 $O(n^3)$
- $X^TX$ 可能不可逆（特征线性相关）

#### 方法二：梯度下降

迭代更新参数：

$$w := w - \alpha \frac{\partial J}{\partial w}$$

其中 $\alpha$ 是学习率。

梯度计算：
$$\frac{\partial J}{\partial w_j} = \frac{2}{m}\sum_{i=1}^{m}(y^{(i)} - \hat{y}^{(i)})x_j^{(i)}$$

### 2.3 三种梯度下降变体

| 类型 | 每次更新使用数据 | 特点 |
|------|------------------|------|
| 批量梯度下降(BGD) | 全部数据 | 收敛稳定，速度慢 |
| 随机梯度下降(SGD) | 单个样本 | 速度快，波动大 |
| 小批量梯度下降(Mini-batch) | 一批数据 | 平衡速度和稳定性 |

## 三、正则化

### 3.1 岭回归 (Ridge, L2正则化)

$$J(w) = \frac{1}{m}\sum_{i=1}^{m}(y^{(i)} - \hat{y}^{(i)})^2 + \lambda\sum_{j=1}^{n}w_j^2$$

**效果**：
- 权重整体变小，但不会为零
- 解决特征共线性问题
- 闭式解：$w = (X^TX + \lambda I)^{-1}X^Ty$（一定可逆）

### 3.2 Lasso回归 (L1正则化)

$$J(w) = \frac{1}{m}\sum_{i=1}^{m}(y^{(i)} - \hat{y}^{(i)})^2 + \lambda\sum_{j=1}^{n}|w_j|$$

**效果**：
- 产生**稀疏解**，部分权重变为0
- 自动特征选择
- 无闭式解，需用坐标下降法

### 3.3 为什么L1产生稀疏解？

**几何解释**：

L1正则化的约束区域是菱形（有尖角），损失函数的等高线与约束区域更容易在顶点处相切，顶点处某些 $w_j = 0$。

**数学解释**：

L1的导数是常数 $\pm\lambda$（与 $w$ 大小无关），当 $w$ 很小时，减去常数可能直接跨过零点。

L2的导数是 $2\lambda w$，当 $w$ 变小时梯度也变小，永远不会到零。

---

## 四、假设条件

线性回归的有效性基于以下假设：

1. **线性关系**：$y$ 与 $x$ 确实存在线性关系
2. **误差独立同分布**：误差 $\epsilon$ 相互独立
3. **误差正态分布**：$\epsilon \sim N(0, \sigma^2)$
4. **同方差性**：误差方差恒定

**违反假设怎么办？**
- 非线性关系 → 多项式回归、特征变换
- 异方差 → 加权最小二乘
- 非正态 → 数据变换（对数、Box-Cox）

---

## 五、优缺点

### 优点
- 简单易懂，计算高效
- 可解释性强（权重反映特征重要性）
- 有闭式解
- 不易过拟合

### 缺点
- 只能拟合线性关系
- 对异常值敏感
- 需要满足假设条件
- 特征共线性影响稳定性

## 六、代码实现

### Python手写实现

```python
import numpy as np

class LinearRegression:
    def __init__(self, learning_rate=0.01, n_iterations=1000):
        self.lr = learning_rate
        self.n_iterations = n_iterations
        self.weights = None
        self.bias = None

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0

        # 梯度下降
        for _ in range(self.n_iterations):
            y_pred = np.dot(X, self.weights) + self.bias

            # 计算梯度
            dw = (1 / n_samples) * np.dot(X.T, (y_pred - y))
            db = (1 / n_samples) * np.sum(y_pred - y)

            # 更新参数
            self.weights -= self.lr * dw
            self.bias -= self.lr * db

    def predict(self, X):
        return np.dot(X, self.weights) + self.bias
```

## 七、面试高频问题

### Q1: 线性回归为什么用MSE不用MAE？

**MSE优势**：
- 凸函数，保证全局最优
- 可导，便于梯度下降
- 对大误差惩罚更大（平方效应）

**MAE劣势**：
- 在零点不可导
- 非光滑，优化困难

### Q2: 如何判断线性回归模型的好坏？

1. **R²决定系数**：解释方差的比例
2. **MSE/RMSE**：预测误差大小
3. **残差图**：检查假设是否满足
4. **特征显著性检验**：t检验、F检验

### Q3: 特征共线性有什么影响？如何解决？

**影响**：
- 权重不稳定
- 难以解释特征重要性
- $(X^TX)$ 近似奇异，求逆不稳定

**解决**：
- 正则化（Ridge）
- PCA降维
- 特征选择
- VIF检验剔除

### Q4: 线性回归 vs 逻辑回归？

| 对比 | 线性回归 | 逻辑回归 |
|------|----------|----------|
| 任务 | 回归 | 分类 |
| 输出 | 连续值 | 概率[0,1] |
| 激活函数 | 无 | Sigmoid |
| 损失函数 | MSE | 交叉熵 |

### Q5: 如何处理非线性关系？

1. **多项式特征**：$x^2, x^3, xy$
2. **特征变换**：对数、指数、根号
3. **分箱(Binning)**：连续转离散
4. **使用非线性模型**：决策树、神经网络
